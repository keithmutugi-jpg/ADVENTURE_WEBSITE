<h1>Keith Safari Adventures</h1>

<p>
Keith Safari Adventures is a simple safari booking web application built using 
HTML and CSS. The website allows users to view safari packages, 
fill in booking details, and store their bookings using Local Storage.
</p>

<h2>Project Purpose</h2>
<p>
The goal of this project is to practice front-end web development by creating 
a functional safari booking system that stores and displays booking data dynamically.
</p>

<h2>Business Logic</h2>
<p>The core booking structure is stored as an object:</p>



<p>
When a user submits the form:
</p>
<ul>
  <li>Booking data is saved into Local Storage</li>
  <li>Saved bookings are displayed dynamically on the page</li>
</ul>

<h2>Technologies Used</h2>
<ul>
  <li>HTML5 – Structure of the website</li>
  <li>CSS3 – Styling and layout design</li>
  <li>Local Storage – Saving booking data in the browser</li>
  <li>Git & GitHub – Version control and project hosting</li>
  <li>VS Code – Code editor used for development</li>
</ul>

<h2>Known Bugs</h2>
<ul>
  <li>Form validation is basic and does not fully prevent incorrect input.</li>
  <li>Duplicate bookings can be added.</li>
  <li>Styling may not be fully responsive on very small screens.</li>
</ul>
